## 📖 About

Figura Cloud is a cloud backend platform designed for Figura Mod users in Minecraft.

The project provides avatar storage, data management, and API services through a centralized PostgreSQL database, helping improve reliability, accessibility, and future scalability for the Figura community.

---

## ❓ Why Figura Cloud?

Many Figura users rely on limited or self-hosted solutions for avatar data management.

Figura Cloud aims to provide:

- ☁️ Reliable cloud-based avatar storage
- 🚀 Faster data access and delivery
- 🔒 Secure data management
- 📡 API support for integrations and tools
- 📈 Scalable infrastructure for future growth
